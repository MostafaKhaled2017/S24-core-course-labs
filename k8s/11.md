## Secrets Management with `Kubectl`

```yaml
$ kubectl create secret generic demo-secret --from-literal=password=sensitive
secret/demo-secret created

$ kubectl describe secrets/demo-secret
Name:         demo-secret
Namespace:    default
Labels:       <none>
Annotations:  <none>

Type:  Opaque

Data
====
password:  9 bytes
```
